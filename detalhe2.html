Analisar o estado físico (danos causados) das pessoas envolvidas nos acidentes é importante para avaliar a gravidade desses acidentes. Porém a base de dados não traz a informação para a maior parte dos casos, sendo que para 2010 e 2008 a falta de informação sobre o estado físico das pessoas acidentadas afeta quase todos os registros. A melhoria no registro dessa informação é um ponto importante para o aprimoramento do sistema de registro de ocorrências.
